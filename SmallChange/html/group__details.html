Detail classes are objects that deliver additional information wherever the SoPrimitiveVertex class is used.

This goes for ray picking results, and it also goes for primitive generation by SoShape classes for the SoCallbackAction class and fallback shape rendering through primitive generation (a useful debugging trick when debugging for instance ray picking).
